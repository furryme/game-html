# #4 记忆碎片 ★★★☆☆

> 排名：#4 / 10 | 交叉评分：美学 ★★★★ / 留存 ★★★ / 玩法 ★★
> 工作量：中-大 | 推荐：叙事主题赛季，建议作为 #1 赛季传说的一个独立赛季

---

## 一句话

每局是一段逐渐清晰的人生回忆，不存在"失败"，方块消除即回忆被拾起

## 核心要解决的问题

如何让方块游戏承载**情感叙事**，而不仅仅是空间规划的快感？

## 核心机制

### 1. "遗忘→拾起"替代"紧张→释放"

- 方块随时间褪色（失去饱和度），褪色方块形状辨识度下降
- 消行不是"消失"而是"回忆被点亮"——消行瞬间方块恢复为鲜艳色彩并短暂绽放
- **不存在"Game Over"**——场地被填满后进入"平静状态"，所有方块缓慢上浮消散

危机感来自"失去"而非"失败"。

### 2. 参数驱动视觉系统

以下参数沿叙事时间轴曲线插值，将逐段制作降为一次性调参：

- **色相偏移** — 回忆的"年代感"变化（冷色调→暖色调）
- **透明度** — 模糊→清晰
- **边缘柔化度** — 遥远的记忆边缘模糊
- **背景叠加层** — 不同回忆场景的背景渐变
- **音频混响参数** — 空旷→温暖

### 3. 叙事曲线

一局约 10-15 分钟，分为 4 个"回忆章节"，每章有不同的视觉色调和旁白：

| 章节 | 色调 | 旁白方向 | 方块行为 |
|------|------|---------|---------|
| 第一章·遗忘 | 灰蓝、低饱和 | "那些被遗忘的......" | 褪色速度最快 |
| 第二章·微光 | 暖黄、渐亮 | "偶尔会闪过一个画面......" | 消行时短暂闪光 |
| 第三章·清晰 | 多彩、高饱和 | "啊，我想起来了......" | 褪色减缓，消行产生涟漪 |
| 终章·完整 | 全彩、发光 | "这就是完整的回忆。" | 方块不再褪色，所有消行产生光粒 |

### 4. 记忆图谱

通关后解锁"记忆图谱"——一张可视化的关系图，显示你已解锁的所有回忆片段如何连接。图谱以节点连线形式展示，类似《Outer Wilds》的知识图谱。

## "Aha" 时刻

把方块游戏的"清空"重新定义为"拾起"——消行不是毁灭，而是记忆被点亮。

## 技术实现要点

- **褪色系统**：每个已放置方块记录放置时间戳，使用 `HSL` 颜色模型，`Saturation` 值随 `(currentTime - placedTime) / decayRate` 递减
- **参数插值**：使用 Catmull-Rom 样条曲线，沿叙事时间轴插值 5 个视觉参数
- **"平静状态"**：场地填满时激活，所有方块逐行向上平移并 `opacity: 0` 消散，约 5 秒完成
- **记忆图谱**：使用 Canvas 2D 绘制节点（已解锁的回忆章节）和连线（章节间关系关系），存储为 JSON 图结构
- **旁白**：使用文本打字机效果 + Web Audio 低频率环境音，无需外部配音

## 与列表中其他创意的关系

- 作为 **#1 赛季传说** 的一个独立赛季运行时，叙事成本可控（只做一季的叙事内容）
- 可与 **#6 腐朽方块** 组合：方块褪色=腐朽，推走腐朽块=遗忘被清理
- 参数驱动系统可与 **#9 声波共鸣** 结合：音频混响参数随章节演变

## 交叉评审中的致命缺陷与讨论

| 评审方 | 致命缺陷 | 回应/缓解 |
|--------|---------|----------|
| gameplay-1（玩法） | 拿走失败等于拿走 Tetris 的核心循环驱动力 | "无失败"消解了紧迫感，但叙事吸引力可以部分补偿；通过"褪色→看不清→需更精准操作"维持机制张力 |
| gameplay-1（玩法） | 叙事内容成本远超预估 | 参数化视觉系统将逐段制作降为一次性调参；建议只做一季而非多季 |
| progression-1（留存） | 叙事内容消费完毕即断崖式流失 | 作为赛季主题而非独立游戏；记忆图谱提供多周目解锁目标 |
| aesthetic-1（美学） | "褪色→看不清→放错→更褪色"恶性循环风险 | 褪色速度精确调平，确保褪色方块仍可辨识形状（只降低饱和度，不降低对比度） |
| fusion-1（创新） | 叙事制作成本（旁白、音乐、视觉、记忆图谱）被低估 | 旁白改为文本打字机效果；音乐为程序化生成（5 个参数驱动），无需外部素材 |
