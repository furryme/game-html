## 维度评分矩阵

### 好玩度（经典组评价标准）
| 维度 | 标准 |
|------|------|
| 上瘾循环 | 是否让人想"再来一局" |
| 重复可玩性 | 多次游玩是否依然有趣 |
| 操作手感 | 输入和反馈的即时满足 |

### 传播力（潮流组评价标准）
| 维度 | 标准 |
|------|------|
| 病毒潜力 | 是否让人想分享给朋友 |
| 一句话解释 | 能否用一句话让人想玩 |
| 截图美学 | 游戏画面是否值得截图 |

### 新意度（融合组评价标准）
| 维度 | 标准 |
|------|------|
| 独特性 | 是否是第一次看到这种玩法 |
| "啊哈"时刻 | 玩家是否会惊呼 |
| 跨界融合 | 是否打破了类别边界 |

| 等级 | 预估时间 | 技术栈 |
|------|---------|--------|

| 中 | 1~3天 | Canvas + Web Audio / 物理模拟 |
| 高 | 3天+ | 复杂物理引擎 / 生态模拟 / AI |
