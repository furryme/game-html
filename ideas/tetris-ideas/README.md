# 俄罗斯方块创意玩法 — Top 4 设计文档

> 来源：8 位专家（4 组 × 2 人）三轮评审收敛
> - Round 1：独立创意生成（40+ 个候选）
> - Round 2：组内讨论与收敛（每组 Top 3 + 完整候选池）
> - Round 3：交叉评审与打分（四维评分 + 致命缺陷挑战）

## 遴选标准

- **不重复**：与现有 `tetris.html`（元素方块）功能区分，避免特殊方块/法术的简单变体
- **可落地**：单文件 HTML/JS 实现，Canvas 2D + 原生 JS
- **差异化**：覆盖机制创新、视觉爽感、社交竞技、社交传播四个维度
- **记忆点**：每个创意都有明确的"啊哈"时刻

### 评选流程
1. **独立生成** — 每位专家生成 5+ 个创意，总计 40+ 候选
2. **组内收敛** — 组内 peer-to-peer 讨论，互相挑刺，选出 Top 3
3. **交叉评审** — 跨组质疑打分，每组从自身维度挑战其他组候选
4. **综合评定** — 汇总四维分数，选出最具价值的 4 个创意
