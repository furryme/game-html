# F4 弦月 — 物理直觉

> 原始概念：投石机/抛体物理，弹道预判的成就感。
> 综合评分：★★★☆☆ | 预估代码量：~800 行 | 开发优先级：P2

---

## 一句话描述

底部充能轨道上轻滑一发，球沿着抛物线穿过移动障碍物的缝隙——弹道不是用来击碎的，是用来累积成画的。

---

## 核心玩法

### 操作（1 个输入）

- **底部固定充能轨道**：手指从轨道中心向任意方向**滑动**
  - 滑动距离 = 力度
  - 滑动方向 = 发射角度
  - 滑动过程中实时显示**弹道预视线**（虚线抛物线）
  - 松开发射
- 关键优势：操作控件固定在底部，不遮挡弹道视野。一个手指一个手势完成所有输入。

### 30 秒体验

1. 打开游戏：深色夜空背景，空中有缓慢移动的障碍带（Lissajous 曲线），右侧有一个目标点
2. 第一次发射：从底部轨道向上滑动，看到弹道预视线穿过障碍缝隙→松开→球沿弧线飞出
3. 命中：球穿过缝隙命中目标，目标**绽放**成粒子（非爆炸），弹道留下一道暖色墨线
4. 墨线缓慢淡去，但 3-5 秒内仍可见。再次发射，新的墨线与旧轨迹叠加
5. 3-5 次发射后：屏幕是一幅由弹道组成的抽象画，而非清空的关卡
6. 所有墨线淡去后，关卡参数微调，进入下一轮

### "校准而非击碎"

与打砖块的核心差异：目标不是摧毁，而是**穿过缝隙到达终点**。球命中障碍物时不碎裂，只是被弹开并留下另一种美丽的弧线——偏离缝隙不是失败，只是弹道变成了另一种风景。

### 弹道即艺术

每次发射后的弹道墨线**缓慢淡去**（而非消失），3-5 次叠加后屏幕形成由弹道组成的抽象画。这将被视为一件临时艺术品，墨线全淡去后才进入下一关。

### 难度递进

- 初期：静止障碍物 + 大缝隙，直觉瞄准即可
- 中期：障碍带缓慢移动，需要把握时机
- 后期：动态风场（1D Perlin Noise 沿 Y 轴），弹道预视线实时受风力修正
- 始终保证解析弹道验证关卡有解

## 解压与哲学

### 解压点

**直觉验证的快感**。"预测 → 验证"的认知循环——大脑天然享受"我猜对了"的确认感。但重新框架为"身体的信任"而非"大脑的计算"——拉弦释放的隐喻从 Angry Birds 的"暴力投掷"转为"放下/信任"。

### 哲学内核

**顺应规律**（射箭哲学）。你不能违背重力，但你可以学会与重力共舞。箭一旦离弦，你唯一能做的就只有信任。暗喻人生的目标感——无法控制所有变量，但通过理解和运用规律，将有限的力量用在最关键的点上。

### "禅"时刻

连续几次调整后，终于找到"完美的一击"——抛体沿着几乎完美的弧线飞过，精准穿过缝隙。此时慢放画面，抛体在空中划出的轨迹留下一道发光的弧线，像烟花一样在空中停留数秒。玩家屏住呼吸，然后长舒一口气。

### 情感弧线

蓄力（紧张）→ 发射（释放）→ 观看弧线（屏息）→ 命中（长舒一口气）→ 欣赏弹道成画（平静）

### 隐喻设计

弹道不可修正——一旦松开，结果由你的判断和世界的规律共同决定。暗喻人生中那些"一旦出手就无法收回"的决定。

## 视觉与声音

### 视觉语言

- **色彩**：暗色背景（深蓝夜空），弹道轨迹为明亮暖色（橙金或暖白），强烈对比
- **节奏**：发射前缓慢蓄力（2-3 秒）→ 发射瞬间短暂加速 → 命中后慢放。张弛有度
- **动效**：命中不做爆炸，而是"绽放"——目标像花朵散开成粒子，缓慢下落消散
- **Wow 效果**：子弹时间命中（0.2x → 0x → 粒子爆发），3 次弹道叠加成一幅抽象画

### 声音设计

- 蓄力：弦乐低频音逐渐升高（不刺耳）
- 发射：短促的"嗖"
- 命中：水晶般清脆回响（singing bowl 音叉）
- 偏离：低沉的嗡鸣（非负面，是"另一种结果"的暗示）
- 整体：参考日本陶板苑（音の泉）自然共鸣

## 技术方案

| 项目 | 方案 |
|------|------|
| 核心算法 | 半隐式 Euler 积分 + 解析弹道预测 |
| 关卡生成 | 参数化生成 + 解析弹道验证有解 |
| 障碍物 | Lissajous 曲线（生成后验证有解） |
| 风场 | 1D Perlin Noise 沿 Y 轴 |
| 弹道预视线 | 100 个采样点，拖拽时实时计算 |
| 弹道渲染 | 离屏 Canvas 累积 + 渐隐动画（alpha 从 1.0 → 0.0，3 秒） |
| 代码量 | 核心 ~480 行 + 基础设施 ~320 行 |
| 移动端适配 | 底部轨道增大 hit area（约 80px 高），竖屏翻转重力方向 |

### 关键技术要点

- **半隐式 Euler 积分器**：2 行代码，能量守恒性接近 Verlet，对恒定加速度场景足够
- **解析弹道验证**：靶位随机放置后，用弹道预测函数生成 100 个采样点检查无障碍路径。仅在关卡生成时运行一次
- **Lissajous 障碍**：参数化曲线 `x = A·sin(at+δ)`, `y = B·sin(bt)`，生成后验证有解，无解则丢弃

## 玩家体验

### 摸鱼伪装度

**中等偏高**。抽象画风 + 深色背景看起来像物理模拟或数据演示。但手指在底部大幅滑动操作较显眼。

### 上手门槛

4 款中最低之一。第一次发射必中（关卡设计保证），给即时正反馈。弹道预视线提供直观引导。

### "再来一局"诱因

"差一点就成"的心理粘性。错过缝隙时的不甘心驱动重试，但偏离不是失败，只是另一种美丽弧线。

### 传播潜力

GIF 或慢放才能传达乐趣，单张截图传播力较弱。关卡参数可编码为 URL hash 适合技术社区。

### 长期留存

- 关卡参数化生成保证无限变化
- 风场增加额外维度
- 弹道成画的累积模式提供视觉奖励

### 遗留问题（需原型验证）

- 滑动充能的方向直觉：手指从轨道中心向"上方"滑动 = 高角度在移动端是否自然？还是"左右滑动控制角度"更直觉？
- **Plan B**：如果"滑动充能"原型验证失败，改用"自动旋转角度指针 + 玩家只控制力度和时机"的降维方案

## 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 积分器 | 半隐式 Euler | Verlet 收益极低，代码多 2 行 |
| 目标 | 穿过缝隙，非击碎 | 与打砖块差异化；校准而非摧毁 |
| 弹道 | 累积成画，非命中清空 | 解压组核心贡献：从测试变为创作 |
| 输入 | 底部固定轨道滑动 | 不遮挡弹道，一个手指完成 |
| 关卡 | 解析验证有解 | 避免无解关卡导致的挫败 |
