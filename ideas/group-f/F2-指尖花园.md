# F2 指尖花园 — 程序生成艺术

> 原始概念：游戏过程就是一件艺术品，每次都是独一无二的画。
> 综合评分：★★★★☆ | 预估代码量：~850 行 | 开发优先级：P3

---

## 一句话描述

在屏幕上轻点"播种"，Simplex Noise 流场引导数百颗粒子自动绘制，你的每一次放置都是一幅不可复制的抽象画。

---

## 核心玩法

### 操作（1 个输入）

- **点击**画布放置"吸引子"，附近的粒子会被引导向该点运动
- 粒子沿流场轨迹运动时留下发光拖尾
- 每次点击自动切换一种画笔颜色（HSL 色轮，低饱和度大地色系）

### 30 秒体验

1. 打开游戏：深色画布上，300 颗粒子沿 Simplex Noise 流场缓慢漂移，留下微弱光痕
2. 第一次点击：放置吸引子，附近粒子向该点汇聚，形成涡旋状星云
3. 第 2-3 次点击：不同位置的吸引子与流场产生干涉，粒子轨迹在两种力之间交织
4. 约 15 秒后：画面涌现出一种"自己都没想到的和谐"——某些轨迹恰好呼应，形成意外的螺旋图案
5. 画局结束：画面定格 2 秒，展示"作品质量"的定性评价（星星等级，非具体分数）

### 游戏模式

**自由创作模式**：无限点击，随时可停，作品自动保存为种子编号。

**挑战种子模式**：每局给定 N 次点击限制（由种子复杂度决定，简单 8-12 次，复杂 15-20 次）。N 次用完后画面定格评分。不满意可随时开始新局。挑战种子解决了"纯创作缺乏目标感"的问题。

## 解压与哲学

### 解压点

**安全区内的探索**。玩家不需要美术功底，只需跟随直觉做选择。系统自动将简单操作转化为优雅的视觉结果。没有"画不好"的焦虑，因为算法保底了美感底线。

### 哲学内核

**过程即目的**（禅宗"一期一会"）。每一幅画都是不可复制的、一次性生成的。游戏结束后画面消散或保存为纪念。当下这一刻的美，正因为不可重现而珍贵。

### "禅"时刻

玩家连续操作若干步后，画面突然出现一种涌现的和谐——某种颜色恰好与之前的线条呼应，形成意外的韵律感。玩家会停下来说"哇，这好像一幅真正的画"。

### 情感弧线

从无意识的点到有意识的引导 → 发现图案的涌现之美 → 退后欣赏整幅作品 → 平静地放下，准备下一幅。

## 视觉与声音

### 视觉语言

- **色彩**：低饱和度大地色系 + 偶尔的明亮点缀色（莫兰迪色系）。深灰背景，粒子轨迹用 `lighter` 混合模式叠加发光
- **节奏**：所有动画缓慢、平滑。粒子运动速度约为手绘感的 0.6 倍速
- **动效**：长曝光效果——粒子轨迹像墨水在宣纸上晕染，逐渐变淡但不完全消失
- **Wow 效果**：涡旋星系——当两个吸引子距离合适时，粒子在两者之间形成壮观的螺旋图案

### 声音设计

- 放置吸引子 = 风铃轻碰（高频但柔和，低通滤波）
- 粒子运动 = 几乎听不见的低频嗡嗡声（类似远处蜂群）
- 画局完成 = 一声远海海浪
- 整体听感：白噪音里的点缀，不突兀

## 技术方案

| 项目 | 方案 |
|------|------|
| 核心算法 | Simplex Noise 2D/3D (FBM) 流场 + 粒子追踪 |
| 粒子数 | 移动端 300 / 桌面端 800 |
| 渲染 | Canvas 2D + 离屏 Canvas 累积 + `lighter` 混合 |
| 评分系统 | 覆盖率(40%) + 颜色层数(35%) + 轨迹长度(25%)，过程中累积计算，~10 行 |
| 噪声预计算 | 256x256 lookup table + 双线性插值 |
| 代码量 | 核心 ~500 行 + 基础设施 ~350 行 |
| 移动端适配 | 触摸放置吸引子，粒子数减少，画布适配屏幕宽度 |

## 玩家体验

### 摸鱼伪装度

**极高**。屏幕看起来像动态设计稿或数据可视化。同事路过会以为是某种设计工具的演示页面。

### 上手门槛

零门槛。打开即玩，不需要教学。第一次点击就能看到粒子被吸引的直观效果。

### "再来一局"诱因

每次随机种子产生不同的流场，每次放置位置不同，画面完全不同。"挑战种子"模式增加目标感，让人想尝试不同策略。

### 传播潜力

每张作品都是天然社交媒体内容。支持导出图片和种子分享码。

### 长期留存

- 随机种子系统保证每次体验不同
- "挑战种子"模式提供可分享的成就
- 元素类型可随版本扩展（不同流场变体）

---

## 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 算法 | 专注流场，不切多算法 | 多算法 = demo，无游戏目标 |
| 评分 | 三维度累积评分 | Sobel 过度工程；定性评价降低公平性质疑 |
| 交互 | 点击放置吸引子 | 长按生长太显眼，鼠标移动轨迹与创意贪吃蛇重叠 |
| 难度 | 挑战种子 + 点击限制 | 纯自由创作缺乏目标感；阶段系统违背"随时可停" |
