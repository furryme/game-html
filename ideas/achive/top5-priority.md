## 1. 反向贪吃蛇 [P0]

**一句话：** 你操控"食物"，躲避 AI 蛇的追击。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐⭐ 经典生存挑战，认知反转 |
| 传播力 | ⭐⭐⭐⭐⭐ "就是那个你当食物的游戏" |
| 新意度 | ⭐⭐⭐⭐ 经典反转，直觉颠覆 |
| 实现难度 | ⭐ 最低，贪吃蛇反向 AI |

**核心操作：** 方向键移动 + 空格放陷阱
**成瘾循环：** 蛇越来越多、越来越快 → "还能苟多久"的生存挑战 → 死亡后看分数 → "再来一局"
**技术实现：** Canvas 2D 渲染 + 简单蛇 AI（BFS/追逐）+ 陷阱机制
**文件：** `games/reverse_snake.html`

## 2. 拆弹专家 [P1]

**一句话：** 扫雷逻辑的炸弹包装，倒计时让每步都是生死抉择。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐⭐ 天然紧张感驱动"再来一局" |
| 传播力 | ⭐⭐⭐⭐ 视觉冲击力强，截图即场景 |
| 新意度 | ⭐⭐⭐ 成熟玩法的全新包装 |
| 实现难度 | ⭐⭐ 低，算法成熟 |

**核心操作：** 点击剪线 + 右键标记
**成瘾循环：** 扫雷推理 → 倒计时压迫 → 成功拆除的巨大成就感 → 下一颗炸弹
**"啊哈"时刻：** 从闹钟拆到核弹，每级难度都有新机制
**技术实现：** 经典扫雷网格算法 + 炸弹视觉（电路板/彩色线）+ 倒计时特效 + 音效
**文件：** `games/demolition_expert.html`

## 3. Bug猎人 [P2]

**一句话：** 游戏故意出 Bug，你来修复——它知道自己是个游戏。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐ 好奇心驱动探索 |
| 传播力 | ⭐⭐⭐⭐⭐ Meta 玩法本身就是传播素材 |
| 新意度 | ⭐⭐⭐⭐⭐ 打破第四面墙，独一无二 |
| 实现难度 | ⭐⭐⭐ 中，DOM 操控+随机 Bug 生成 |

**核心操作：** 发现异常 → 点击修复
**成瘾循环：** 每个 Bug 都出乎意料 → 修复后解锁荒诞叙事 → "下一个会怎样"的好奇
**"啊哈"时刻：** 游戏对你说"你又修好了一个，但我马上给你造个新的"
**技术实现：** DOM/CSS 操作制造"Bug"效果（按钮跑偏、颜色反转、文字乱码）+ 随机生成器 + 最终 Boss "404"
**文件：** `games/bug_hunter.html`

## 4. 轮回密码 [P3]

**一句话：** 被困在无限重置的 24 小时，每次死亡只能记住一条信息。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐⭐ 死亡是机制而非惩罚 |
| 传播力 | ⭐⭐⭐⭐ "那个会重置的游戏" |
| 新意度 | ⭐⭐⭐⭐⭐ 玩法即叙事，解密=揭开真相 |
| 实现难度 | ⭐⭐⭐ 中，状态继承+网格解密 |

**核心操作：** 点击格子 + 放置标记（每轮只能放一个）
**成瘾循环：** 扫雷式推理 → 死亡重置但保留标记 → 带着记忆回来，离真相更近 → 每一轮都有进展感
**"啊哈"时刻：** 场景细节每轮微妙变化，暗示循环中的"另一个人"其实是你自己
**技术实现：** Canvas 网格 + 循环状态管理（标记继承）+ 场景细节变化 + 日志叙事系统
**文件：** `games/loop_cipher.html`

## 5. 时间导演 [P4]

**一句话：** 录制自己的操作，和过去的自己协作通关。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐⭐ 协作解谜的成就感 |
| 传播力 | ⭐⭐⭐⭐ "站在自己肩上"的截图 |
| 新意度 | ⭐⭐⭐⭐⭐ 创意独一无二 |
| 实现难度 | ⭐⭐⭐⭐ 较高，多实体回放 |

**核心操作：** 方向键跳跃 + 录制/播放 + 最多 3 轨
**成瘾循环：** 观察关卡 → 录制第一轨 → 发现需要配合 → 录制第二轨卡住时机 → 第三轨完成通关
**"啊哈"时刻：** 意识到"过去的自己"可以当工具——站在自己肩上跳得更高，三个自己协作按下三个开关
**技术实现：** 操作录制/回放系统 + 多实体并行渲染 + 平台跳跃物理 + 关卡编辑器
**文件：** `games/time_director.html`

## 为什么是这五个？

1. **覆盖五种类型：** 生存、解谜、Meta、策略、平台跳跃 — 互不重复
2. **实现梯度合理：** 从半天到三天，逐步建立信心和技术储备
3. **传播组合拳：** 既有极简病毒（反向贪吃蛇），也有口碑之作（时间导演）
4. **单文件 HTML 可行：** 全部 Canvas 2D + JS，无需构建工具
5. **有灵魂：** 不只是机制堆砌，每个都有让人记住的那个"点"
