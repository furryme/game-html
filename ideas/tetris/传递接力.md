# 传递接力

**一句话：** 将棋盘状态编码为分享码传给下一人接力，3-5 人成链，"这局拜托你了"的社交叙事驱动传播。

| 维度 | 评分 |
|------|------|
| 好玩度 | ⭐⭐⭐⭐ "救场英雄/坑队友"的情感张力，每局都是不同起点的新挑战 |
| 传播力 | ⭐⭐⭐⭐⭐ "那个可以传方块的俄罗斯方块"，分享即玩法 |
| 新意度 | ⭐⭐⭐⭐ 将异步社交融入方块，"托付→接力→拯救/背叛"的情感弧线 |
| 实现难度 | ⭐⭐ 低，棋盘状态编码 + URL 分享 |

## 核心机制详解

### 接力流程
1. 玩家开始一局 Tetris，正常玩到觉得"该传了"
2. 点击"传递"按钮，棋盘状态被编码为短分享码
3. 分享码通过复制/微信/短信发送给好友
4. 好友打开链接（或粘贴分享码），看到前人的棋盘继续玩
5. 3-5 人成链，中途 Game Over 则链断裂，显示接力记录

### 状态编码
- 棋盘 10×20 = 200 格，每格 2bit（0=空，1-7=方块类型）= 400bit = 50 字节
- 附加数据：分数（2 字节）、接力次数（1 字节）、前人名（4 字节）
- Base62 编码压缩，分享码长度约 80-100 字符
- **URL 方案**：`?board=...&score=...&chain=...`
- 纯文本分享码也支持，复制粘贴即可

### 开局检测
- 收到分享码后，自动检测棋盘是否"可玩"
- **可玩标准**：棋盘顶部有空格放置所有 7 种方块中的至少 3 种
- 不可玩的棋盘会提示"此局已无解，开始新链"，并保留前人的分数记录

### 接力记录
- 每局底部显示接力链：`A → B → C → 你`
- 每人显示：名字（首字母）、消行数、存活时间
- Game Over 后显示完整排行榜："你救了 3 个人，最终消了 42 行"
- **复活机制**：每局有一次"求助"机会，点击后随机消除 1 行并生成分享码

### AI 接力兜底
- 单人模式下，"传递"给 AI 接力
- AI 有三种人格：
  - **救场型**：优先消行，给你留好棋盘
  - **坑人型**：故意堆高边缘，留一个死角
  - **佛系型**：随机放置，结果看运气
- AI 接力后显示"AI 消了 X 行，传回给你了"

## 成瘾循环

玩 Tetris → 快不行了点传递 → 好友收到接力 → 好友也传给下一个人 → 你收到别人的棋盘 → 看到前人留下的烂摊子/好棋局 → 继续玩 → 循环

核心张力：**"前人留了什么？是在求助还是在坑我？"** —— 社交叙事驱动而非纯竞技

## "啊哈"时刻

1. **第一次收到接力**：看到别人堆了半屏的棋盘，发现底部刚好能消 3 行
2. **救场英雄**：前人留下一个几乎封顶的棋盘，你消了关键一行救活整局
3. **坑队友**：你传给朋友的棋盘底部有个隐藏的死角，朋友 30 秒后 Game Over
4. **长链成就**：你的接力链达到 5 人，总分 100 行，分享海报上显示完整链条

## 视觉/音效设计

### 视觉
- **传递动画**：当前棋盘右侧滑出，新棋盘从左侧滑入，中间有"传递"特效
- **接力链 UI**：底部横向滚动的人头像链，每个人头是一个圆圈+首字母
- **接收棋盘**：前人的方块用暗色显示，你新放的方块用亮色，区分新旧
- **分享码弹窗**：居中卡片，包含分享码、复制按钮、"开始新链"按钮
- **Game Over 画面**：显示完整接力记录，每人一行（名字、消行数、贡献百分比）
- **AI 人格标识**：AI 接力时显示小人图标+表情（😊 救场 / 😈 坑人 / 😐 佛系）

### 音效
- **传递音效**：类似击掌的"啪"声
- **接收音效**：收到接力时的"叮铃"通知声
- **消行音效**：标准 Tetris 消除声，但接力首消有庆祝音效
- **Game Over**：根据接力链长度变化（短链=悲伤、长链=庆祝）
- **BGM**：轻松的电子乐，接力时有 2 秒过渡音乐

  let bits = '';
  for (let y = 0; y < 20; y++) {
    for (let x = 0; x < 10; x++) {
      bits += board[y][x].toString(2).padStart(2, '0');
    }
  }
  // 附加数据
  bits += score.toString(2).padStart(16, '0');
  bits += chainCount.toString(2).padStart(8, '0');

### 分享方式
- **URL 参数**：`https://.../tetris_relay.html?board=ABC123...&score=4200&chain=3`
- **复制分享码**：纯文本，可短信/微信发送
- **系统分享**：`navigator.share()` API，支持 iOS/Android 原生分享
- **AI 接力**：无需分享，点击按钮直接生成 AI 棋盘

### 可玩性检测
```javascript
function isPlayable(board) {
  const emptyCells = getEmptyCells(board);
  let placeable = 0;
  for (const shape of TETROMINOS) {
    if (canPlaceAnywhere(board, shape)) {
      placeable++;
    }
  }
  return placeable >= 3; // 至少能放3种方块
}
```

### 移动端适配
- 全屏单棋盘，接力链折叠到底部
- 分享按钮大而醒目，位于暂停菜单顶部
- 虚拟方向键 + 旋转/硬降按钮
- 分享码弹窗支持长按复制

### 数据持久化
- localStorage 存储历史接力记录（非必需）
- 接力链统计数据：最长链、总消行、救场次数
- 隐私：不存储真实姓名，只显示首字母

## 预估代码量和开发优先级

- **预估代码量**：~900 行
- **开发优先级**：P3（实现简单但传播价值高，适合作为"轻量"作品）
- **预估工期**：1 天
- **文件**：`games/tetris_relay.html`

## 与现有 tetris.html 的关系

**独立文件**。核心 Tetris 逻辑可复用，但状态编码、分享系统、接力记录是全新功能。基础方块渲染/输入/消行可提取为共享模块。
