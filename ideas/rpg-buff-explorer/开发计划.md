# RPG Buff 探险者 — 开发计划

## Context

从零开始开发像素风地城探险 Roguelite RPG。已有 8 份设计文档 ~4900 行（经过多专家评审），定义完整。

**MVP 范围**：1 职业（战士）+ 3 层地城 + buff 3 选 1 + Boss 规则战 + 永久成长（灵魂碎片 + 天赋）+ localStorage 存档。

**目标文件**：`ideas/rpg-buff-explorer/游戏.html`（稳定后移入 `web/games/`）

---

## 整体架构

```
单文件 HTML
├── Canvas 层: 地图渲染 / 精灵绘制 / 粒子特效 / 战斗画面
├── HTML 浮层: HUD / 按钮 / 弹窗 / 进度条 / 战斗日志
└── localStorage: buff 解锁进度 / 永久天赋 / 统计数据
```

**核心循环**：探索地城 → 踩到敌人进入回合制战斗 → 获得装备/金币/EXP → Boss 规则战 → 层前 buff 3 选 1 → 下一层 → 通关/死亡 → 永久成长

**脚本分块顺序**：
```
[DATA] → [STATE] → [UTILITIES] → [BUFF] → [EQUIPMENT] → [GENERATOR] → [FOG_OF_WAR] → [MOVEMENT] → [COMBAT] → [SHOP] → [ECONOMY] → [PERSISTENCE] → [RENDER] → [UI] → [INPUT] → [FX] → [AUDIO]
```

## P1 — 核心循环骨架

### 地图生成
- BLOB 膨胀 + MST 走廊 + 20% 冗余环路
- 40×30 格，每层 5-8 房间
- 3 层主题配色（青苔回廊/骨骸 Crypt/虚空核心）
- 环境 debuff

### 房间系统
- 6 种：combat / loot / trap / resting / shrine / empty
- 12 种敌人按层分配

### 移动与碰撞
- WASD/方向键
- wall → enemy → item → trap → stairs
- 战争迷雾 180 射线 FOV
- 小地图 120×90px

### 回合制战斗
- 回合开始 → 先手判定 → 玩家行动 → 敌人行动 → 结算
- 普通攻击 + 防御姿态
- 伤害公式：`max(1, ATK × 倍率 × buff倍率 - DEF × DEF/(DEF+100))`
- 敌人 AI：加权随机

### 等级与属性
- 1-10 级，每局从 1 级开始
- HP / ATK / DEF / SPD / CRIT

## P2 — Buff + 技能 + 装备

### Buff 系统
- 8 默认解锁 buff（声明式数据）
- recalcBuffStats() 聚合
- 层前 3 选 1 UI

### 技能
- 10 个：斩击/重击/连斩/防御/战吼/旋风斩/吸血/破甲/战狂/死亡之舞
- MP + CD 系统

### 装备
- 3 栏位 × 3 稀有度 × 随机词缀

## P3 — Boss + 永久成长

### Boss 规则系统
- 10 个可复用规则模块

- HP 阈值触发规则叠加

- 24 buff 解锁进度
- localStorage 持久化
