# RPG 成长系统设计

## 全景概览

```
                          EXP
                           │
              ┌────────────┼────────────┐
              ↓            ↓            ↓
           等级成长     技能点解锁     装备掉落
              │            │            │
              └────────────┼────────────┘
                           ↓
                      角色战斗力
                           │
              ┌────────────┼────────────┐
              ↓            ↓            ↓
           职业特性      buff 加成     技能组合
              │            │            │
              └────────────┼────────────┘
                           ↓
                    地城推进深度
```

**成长不是一条线而是一个网状结构**——等级、装备、技能树、buff 四条线互相交织，玩家每层地城前都有选择要做。

## 一、等级与基础属性

### 等级曲线

| 等级 | 所需 EXP | 累计 EXP | HP | ATK | DEF | SPD |
|------|----------|----------|----|质5|------|
| 1 | 0 | 0 | 100 | 10 | 5 | 10 |
| 2 | 50 | 50 | 120 | 12 | 6 | 11 |
| 3 | 100 | 150 | 150 | 15 | 8 | 12 |
| 4 | 150 | 300 | 190 | 18 | 10 | 13 |
| 5 | 200 | 500 | 240 | 22 | 13 | 14 |
| ... | ... | ... | ... | ... | ... | ... |
| 20 | 2000 | ~25000 | 800 | 80 | 45 | 30 |

**EXP 获取**：
- 击败敌人：基础 EXP × 等级倍率
- 宝箱：固定 EXP
- 隐藏房间：大量 EXP + 特殊道具
- 层数通关奖励：50 × 层数

**属性成长公式**：
```
HP = 100 + 等级 × 20 × 职业系数
ATK = 10 + 等级 × 3 × 职业系数
DEF = 5 + 等级 × 2 × 职业系数
SPD = 10 + 等级 × 1 × 职业系数
```

### 属性含义

| 属性 | 作用 | 计算公式 |
|------|------|----------|
| HP | 生命值 | 归零则死亡 |
| ATK | 攻击力 | 伤害 = ATK × 技能倍率 - 目标 DEF |
| DEF | 防御力 | 减伤 = DEF / (DEF + 50)  |
| SPD | 速度 | 决定先手概率 = 我方SPD / (我方SPD + 敌方SPD) |
| CRIT | 暴击率 | 基础 5%，装备/buff 叠加 |
| LUCK | 幸运 | 影响掉落率、闪避率 |

## 二、职业分支

### 三职业设计

```
              ┌── 狂战士（ATK 爆发）
              │
      战士 ───┼── 骑士（均衡 / 坦克）
      │       └── 圣骑士（HP / 团队支援）
      │
职业 ─── 法师 ───├── 元素师（范围伤害）
      │          └── 奥术师（单体穿透）
      │
      └── 游侠 ───├── 刺客（先手 / 暴击）
                 └── 猎人（远程 / 持续）
```

### 职业特性

| 职业 | 核心属性 | 被动特性 | 专属技能类型 |
|------|----------|----------|-------------|
| 狂战士 | ATK 3x | 攻击越低伤害越高（嗜血） | 自残换输出 |
| 骑士 | SPD 1.5x, DEF 1.5x | 格挡概率 10% | 防御反击 |
| 圣骑士 | HP 3x | 每回合恢复 5% HP | 治疗 / 增益 |
| 元素师 | ATK 3x | 技能附带元素效果 | AOE / 连锁 |
| 奥术师 | ATK 2.5x | 无视 30% DEF | 单体爆发 |
| 刺客 | CRIT 20%, SPD 2x | 先手必暴击 | 瞬杀 / 闪避 |
| 猎人 | SPD 1.5x | 远程攻击无视距离 | 陷阱 / 持续伤害 |

### 职业解锁

- 初始选一个基础职业（战士 / 法师 / 游侠）
- 等级 5 解锁专精分支
- 专精切换消耗"命运碎片"（稀有道具，不可逆选择——增加决策重量感）

## 三、装备系统

### 装备栏位

```
     [头盔]
    /      \
[武器]--[护甲]--[饰品]
    \      /
     [靴子]
```

5 个栏位：头盔、武器、护甲、靴子、饰品。

### 装备稀有度

| 稀有度 | 颜色 | 属性数量 | 出现率 |
|--------|------|----------|--------|
| 普通 | 白 | 1 条 | 50% |
| 优秀 | 绿 | 2 条 | 30% |
| 稀有 | 蓝 | 2-3 条 | 15% |
| 史诗 | 紫 | 3-4 条 + 特效 | 4% |
| 传说 | 橙 | 4-5 条 + 强力特效 | 1% |

### 装备属性池

```javascript
const AFFIXES = {
  prefix: {
    "强化的": { stat: "ATK", range: [5, 30] },
    "坚硬的": { stat: "DEF", range: [5, 25] },
    "迅捷的": { stat: "SPD", range: [3, 15] },
    "沉重的": { stat: "ATK", range: [10, 40], penalty: { stat: "SPD", value: -3 } },
    "淬毒的": { stat: "CRIT", range: [5, 20] },
  },
  suffix: {
    "之力": { stat: "HP", range: [20, 100] },
    "之护": { stat: "DEF", range: [5, 20] },
    "之速": { stat: "SPD", range: [3, 10] },
    "之魂": { stat: "LUCK", range: [5, 15] },
  },
};
```

### 装备生成

```javascript
function generateEquipment(floor, slot) {
  const rarity = rollRarity();
  const base = BASE_EQUIPMENT[slot][rarity];
  const prefix = pickRandom(AFFIXES.prefix);
  const suffix = pickRandom(AFFIXES.suffix);

  return {
    name: `${prefix}${base.name}${suffix}`,
    slot,
    rarity,
    stats: {
      [prefix.stat]: roll(prefix.range) * floorMultiplier(floor),
      [suffix.stat]: roll(suffix.range) * floorMultiplier(floor),
    },
    effect: rarity >= "epic" ? rollEpicEffect() : null,
  };
}
```

**层数倍率**：每深一层，装备属性 +20%。保证深层地城有换装动力。

### 装备管理

- 背包上限 20 格
- 战场上可即时更换装备（属性立即生效）
- 重复装备自动分解为"强化石"
- 强化石可在 NPC 处出售或用于强化

## 四、技能树

### 技能树结构

每个专精有一棵独立技能树，分三条支线：

```
              [核心被动] ← 必点
             /          \
    [进攻支线]      [防御支线]
     ↓               ↓
   伤害技能       生存技能
     │               │
     └───────┬───────┘
             ↓
       [终极技能] ← 需要两条支线都点亮
```

### 狂战士技能树（示例）

```
进攻支线：                    防御支线：
┌─────────┐                 ┌─────────┐
│ 斩击     │ 5 SP            │ 战吼     │ 5 SP
│ ATK×1.5  │                 │ 本回合DEF+50% │
└────┬─────┘                 └────┬─────┘
     │                             │
     ▼                             ▼
┌─────────┐                 ┌─────────┐
│ 旋风斩   │ 10 SP           │ 怒血护甲 │ 10 SP
│ ATK×2.0  │                 │ 首次受击减伤80% │
│ 范围     │                 └────┬─────┘
└────┬─────┘                      │
     └──────────┬─────────────────┘
                ▼
       ┌──────────────┐
       │ 死亡之舞      │ 20 SP
       │ 10秒内 ATK×3  │
       │ 结束后自毁     │
       │（复活消耗 1×命）│
       └──────────────┘
```

### 技能点获取

- 升级获得 2 SP
- boss 战胜利获得 1 SP
- 通关地城获得 3 SP

### 技能重置

- "命运之书"道具（稀有）可重置全部技能点
- 地城入口 NPC 收费重置（消耗金币）

## 五、buff 联动

### buff 框架接入点

参考 [buff 选择框架](../buff选择框架.md)，RPG 中的 buff 表现为"层前祝福"：

```
进入地城 → 层前大厅 → 选择 3 个 buff → 进入战斗
```

### buff 与成长系统的交互

| 交互类型 | 说明 | 示例 |
|----------|------|------|
| buff × 属性 | buff 直接加成基础属性 | "巨人之躯" HP+50% |
| buff × 技能 | buff 解锁被动技能效果 | "烈焰附魔" 攻击附带灼烧 |
| buff × 装备 | buff 改变装备生成概率 | "寻宝者" 传说装备率 +0.5% |
| buff × 职业 | buff 对特定职业有额外效果 | "狂怒" 狂战士 ATK 加成翻倍 |
| buff × buff | 组合触发隐藏效果 | "火" + "风" = 烈焰风暴 |

### buff 分类

```javascript
const RPG_BUFFS = [
  {
    id: "berserks_blessing",
    name: "狂战士的祝福",
    description: "ATK +30%，但 HP -20%",
    icon: "⚔️",
    rarity: "rare",
    tags: ["attack"],
    effect: (state) => {
      state.atkMultiplier *= 1.3;
      state.hpMultiplier *= 0.8;
    },
    synergy: {
      class: "berserker",  // 狂战士额外加成
      bonus: (state) => { state.atkMultiplier *= 1.2; },
    },
  },
  {
    id: "flame_aura",
    name: "烈焰光环",
    description: "攻击附带灼烧（每回合 10 伤害）",
    icon: "🔥",
    rarity: "legendary",
    tags: ["fire", "dot"],
    effect: (state) => { state.onAttack.push(target => target.addBurn(10)); },
    synergy: {
      tags: ["wind"],  // 与风 buff 组合
      bonus: () => { /* 触发烈焰风暴 */ },
    },
  },
];
```

### buff 组合表

| buff A | buff B | 组合效果 |
|--------|--------|----------|
| 🔥 烈焰光环 | 💨 风暴之力 | 烈焰风暴：攻击附带范围灼烧 |
| 🛡️ 铁壁 | 💀 死亡之拥 | 反击护盾：受到伤害时反弹 50% |
| ⚡ 迅捷 | 🏹 精准射击 | 二连射：每回合可攻击两次 |
| 💰 贪婪 | 🔮 预言之眼 | 宝藏猎人：宝箱开出双倍物品 |

## 六、地城推进与难度曲线

### 地城结构

```
地城入口
  ↓
第 1 层 ── 3 房间 + 1 宝箱 + boss → 层前 buff 选择
  ↓
第 2 层 ── 4 房间 + 1 宝箱 + 1 NPC + boss
  ↓
第 3 层 ── 5 房间 + 1 宝箱 + 1 隐藏房间 + boss
  ↓
...
  ↓
第 5 层 ── 最终 boss
```

### 敌人难度缩放

```
敌人ATK = 基础值 × (1 + 0.3 × 层数) × 地图倍率
敌人HP  = 基础值 × (1 + 0.2 × 层数) × 地图倍率
敌人EXP = 基础值 × (1 + 0.5 × 层数)
```

### 玩家成长速度匹配

目标：每层 boss 战玩家需全力以赴但能胜。

```
玩家期望战力(层N) ≈ boss战力(层N) × 1.2
```

1.2 倍是"有压力但能赢"的安全区间，通过 buff + 装备浮动。

## 七、死亡与重生

### 死亡惩罚

- 死亡后回到地城入口
- 保留：等级、属性、技能树、已解锁 buff
- 丢失：本局装备、金币、未使用的消耗品
- 保留一层前大厅选择的 buff（罗格-lite 的"永恒遗物"机制）

### 永久成长（元进度）

```
通关地城 → 获得"灵魂碎片" → 升级永久天赋树
```

**永久天赋树**（独立于职业技能树）：

```
              [生命涌动] HP+20
             /
      [初始金币+50]
      /
[背包+5格]────────[起始装备栏+1]
```

- 永久天赋用"灵魂碎片"解锁（通关获得）
- 每局游戏从永久天赋起步
- 这是 Roguelite 的核心循环：即使死亡也在变强

## 九、数值平衡框架

### 战斗伤害计算

```
伤害 = max(1, 基础ATK × 技能倍率 × buff倍率 - 目标DEF × 减伤系数)
减伤系数 = 目标DEF / (目标DEF + 50)
暴击伤害 = 伤害 × 1.5
```

### 经济循环

```
战斗 → 金币 → NPC 商店 → 药水 / 装备 / 技能重置
                      → 强化师 → 装备强化
                     → 情报贩子 → 地图提示
```

### 资源稀缺性

| 资源 | 获取方式 | 用途 | 稀缺程度 |
|------|----------|------|----------|
| 金币 | 战斗 / 宝箱 | 药水、装备、服务 | 中等 |
| 药水 | 商店购买 / 宝箱 | 恢复 HP | 每层 1-2 个 |
| 命运碎片 | boss 掉落 | 重置技能树 | 极稀有 |
| 灵魂碎片 | 通关地城 | 永久天赋 | 稀有 |
| 强化石 | 分解装备 | 装备强化 | 充裕 |
